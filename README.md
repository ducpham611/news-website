## Các chức năng chính

- CRUD tin tức, bình luận, tài khoản, phản hồi
- Thống kê các thông tin dưới dạng biểu đồ (dành cho phía admin)



## Công nghệ sử dụng

- Front-end: HTML, CSS, Javascript
- Back-end: PHP (Laravel)
- Database: MySQL
